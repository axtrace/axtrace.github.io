# Настройка Яндекс.Метрики в Diplodoc

## Конфигурация

Яндекс.Метрика настроена через встроенную поддержку Diplodoc в файле `.yfm`:

```yaml
apply-presets: true
varsPreset: external

vars:
  metrika: 54428956
```

## Метаданные для контентной аналитики

Для корректной работы отчетов по контенту в Яндекс.Метрике, каждая страница должна содержать метаданные OpenGraph в формате:

```yaml
---
metadata:
  - property: 'og:type'
    content: 'article'
  - property: 'og:title'
    content: 'Название статьи'
  - property: 'og:description'
    content: 'Описание статьи'
  - property: 'og:url'
    content: 'https://axtrace.github.io/path/to/page'
---
```

## Сборка проекта

При сборке проекта Diplodoc автоматически добавит код счетчика Яндекс.Метрики во все HTML страницы:

```bash
npm run build:docs
```

## Доступные отчеты

В Яндекс.Метрике доступны следующие отчеты по контенту:

- Статистика по материалам, рубрикам, авторам, тематикам
- Статистика по доскроллам и дочитыванию
- Время на чтение статьи
- Динамика просмотра статей
- Доля материалов с дочитыванием
- Процент пользователей, которые перешли на другие страницы сайта
- Доля просмотров материалов с доскроллом
- Показатели вовлечённости и посещаемость рубрик
