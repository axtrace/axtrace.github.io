# Заповедник

**Автор:** Сергей Довлатов

## Отзыв

Прочитал "Заповедник" Довлатова и открыл для себя этого автора.

Больше всего понравился стиль - ироничный, лаконичный, с тонким юмором и глубокими наблюдениями о жизни. Довлатов умеет в нескольких словах передать целую гамму чувств и создать яркий образ.

Повесть автобиографична - рассказывает о работе экскурсоводом в Пушкинском заповеднике. Через призму этого опыта автор размышляет о жизни, любви, творчестве и человеческих отношениях.

## Цитаты

Особенно запавшие в душу цитаты:

{% note info "О жизни и мужестве" %}

> Ведь кошмар и безнадежность - еще не самое плохое. Самое ужасное - хаос...

> Слабые люди преодолевают жизнь, мужественные - осваивают...

{% endnote %}

{% note tip "О любви" %}

> Татьяна взошла над моей жизнью, как утренняя заря. То есть спокойно, красиво, не возбуждая чрезмерных эмоций.

> Хорошего человека любить неинтересно...

> Любовь - это для молодежи. Для военнослужащих и спортсменов... А тут все гораздо сложнее. Тут уже не любовь, а судьба...

{% endnote %}

{% note warning "О творчестве и самоощущении" %}

> Я был - одновременно - непризнанным гением и страшным халтурщиком.

> Единственная честная дорога - это путь ошибок, разочарований и надежд.

> Жизнь - есть выявление собственным опытом границ добра и зла...

{% endnote %}
