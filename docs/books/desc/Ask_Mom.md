# После прочтения

- Важность ведения заметок во время общения с клиентами.
	  - **Ведение заметок:** автор рекомендует делать заметки во время встреч, используя различные носители информации (карточки, стикеры, блокнот). Он подчёркивает важность возможности сортировки, комбинирования и упорядочивания заметок.
	- **Использование символов:** предлагается заменять слова символами для ускорения записи. Автор использует 12 основных символов, но рекомендует читателям разработать свой собственный «язык» символов.
- Необходимость анализа полученной информации вместе с командой.
- Алгоритм общения с клиентами, состоящий из пяти элементов: «Видение / Формирование / Слабости / Значимость / Просьба» (можно запомнить с помощью мнемонической фразы «Вижу Форму — Сразу Запрошу Помощь» (ВФСЗП)).
- Подход к общению, направленный на отбрасывание формальности и неловкости, чтобы научиться задавать правильные вопросы.
- - **Работа с обязательствами:** важно добиваться твёрдых обязательств от клиентов и фиксировать последующие шаги.
