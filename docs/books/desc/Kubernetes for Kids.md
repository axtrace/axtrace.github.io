# Цитаты

> «Kubernetes» — по-гречески, капитан корабля. От него произошли такие слова, как «кибернетика» и «губернатор». Kubernetes создан для управления сотнями и тысячами контейнеров.

> Контроллер репликации содержит шаблон модуля, который можно копировать любое количество раз.
